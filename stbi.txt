Ya, tentu! Ada beberapa database musik besar dan terbuka yang bisa kamu gunakan tergantung pada kebutuhan (riset, proyek machine learning, aplikasi musik, dsb). Berikut beberapa sumber populer dan lengkap yang bisa kamu pertimbangkan 👇
🎧 1. MusicBrainz

    🔗 Website: https://musicbrainz.org

📦 Deskripsi: Salah satu open music database terbesar dan paling lengkap.

🧠 Isi: Judul lagu, artis, album, tanggal rilis, label, genre, metadata rekaman, dan relasi antar entitas.

💾 Akses data:

    API: MusicBrainz API

Dump data lengkap (PostgreSQL): Data Downloads
🎼 2. Discogs

    🔗 Website: https://discogs.com

📦 Deskripsi: Database kolektor musik global dengan fokus pada rilisan fisik (vinyl, CD, kaset).

🧠 Isi: Artis, album, label, tahun, genre, gaya musik, format rilisan, katalog label.

💾 Akses data:

    API publik: Discogs API

Data dump bulanan (CSV/XML): Discogs Data
🎵 3. Last.fm

    🔗 Website: https://www.last.fm

📦 Deskripsi: Fokus pada perilaku pendengar dan metadata musik.

🧠 Isi: Judul lagu, artis, album, genre (tag), popularitas, dan statistik pemutaran.

💾 Akses data:

    API: Last.fm API

🎤 4. TheAudioDB

    🔗 Website: https://www.theaudiodb.com

📦 Deskripsi: Database komunitas berisi metadata artis, album, dan lagu — sering dipakai untuk aplikasi media.

🧠 Isi: Artis, biografi, album, tahun rilis, genre, mood, thumbnail, dan artwork.

💾 Akses data:

    API: TheAudioDB API

🧩 5. Spotify Web API

    🔗 Website: https://developer.spotify.com/documentation/web-api

    📦 Deskripsi: Tidak open database penuh, tapi API-nya sangat kaya.

    🧠 Isi: Lagu, artis, album, genre, popularitas, audio features (tempo, energy, valence, dsb).

    💾 Akses data:

        Perlu registrasi developer (gratis).

        Cocok untuk riset musik dan machine learning.

Ya, kamu bisa mendapatkan file CSV (atau format mirip seperti TSV/JSON) dari beberapa database musik besar — tapi caranya berbeda tergantung sumbernya. Berikut penjelasan detailnya 👇
🎧 1. MusicBrainz

🗂️ Format: Database dump (PostgreSQL), tapi bisa diubah ke CSV.

Cara:

    Unduh dump resmi di sini:
    🔗 https://musicbrainz.org/doc/MusicBrainz_Database/Download

File-nya berupa .tar.bz2 berisi tabel PostgreSQL.

Kamu bisa memuatnya ke PostgreSQL, lalu ekspor tabel jadi CSV:

    \COPY artist TO 'artists.csv' DELIMITER ',' CSV HEADER;

💡 Ada juga mirror tidak resmi yang sudah menyediakan MusicBrainz dalam format CSV (bisa saya bantu carikan yang terbaru kalau kamu mau).
💿 2. Discogs

🗂️ Format: XML atau JSON (bisa dikonversi ke CSV).

Cara:

    Download dump bulanan di sini:
    🔗 https://data.discogs.com/

File besar (beberapa GB) berisi:

    discogs_YYYYMMDD_artists.xml.gz

    discogs_YYYYMMDD_releases.xml.gz

    discogs_YYYYMMDD_labels.xml.gz

Bisa diubah ke CSV pakai Python, misalnya:

    import pandas as pd
    import xml.etree.ElementTree as ET

    tree = ET.parse('discogs_releases.xml')
    root = tree.getroot()

    data = []
    for release in root.findall('release'):
        title = release.find('title').text if release.find('title') is not None else ''
        artist = release.find('artists/artist/name').text if release.find('artists/artist/name') is not None else ''
        year = release.find('year').text if release.find('year') is not None else ''
        data.append([title, artist, year])

    df = pd.DataFrame(data, columns=['title', 'artist', 'year'])
    df.to_csv('discogs_releases.csv', index=False)

🎤 3. TheAudioDB

🗂️ Format: JSON dari API, bisa langsung diubah ke CSV.

Contoh:

import requests
import pandas as pd

r = requests.get('https://theaudiodb.com/api/v1/json/2/search.php?s=metallica')
data = r.json()['artists']
df = pd.DataFrame(data)
df.to_csv('theaudiodb_artists.csv', index=False)

🎵 4. Last.fm

🗂️ Format: JSON dari API, bisa dikonversi ke CSV seperti di atas.
🎶 5. Spotify API

🗂️ Format: JSON hasil API, bisa diubah ke CSV juga.

Contoh kecil:

import spotipy
from spotipy.oauth2 import SpotifyClientCredentials
import pandas as pd

sp = spotipy.Spotify(auth_manager=SpotifyClientCredentials(client_id="YOUR_ID", client_secret="YOUR_SECRET"))
results = sp.search(q='genre:rock', type='track', limit=50)

tracks = []
for item in results['tracks']['items']:
    tracks.append({
        'title': item['name'],
        'artist': item['artists'][0]['name'],
        'album': item['album']['name'],
        'release_date': item['album']['release_date']
    })

pd.DataFrame(tracks).to_csv('spotify_tracks.csv', index=False)
